# OpenGL-Room
A course project using OpenGL to construct a indoor scene.

# 使用方法
## 按键
- w,s,a,d,z,c	进行视点的运动
- g		开启/关闭碰撞检测
- p		截图至pic文件夹下
- ,(<)		视野放大
- .(>)		视野缩小
- /(?)		视野还原
- 1,2		选择所要添加的物体类型
- =(+)		添加物体
- -(_)		删除物体
- x		旋转物体
- i,k,j,l,n,m	物体平移控制
- t		物体纹理控制
- y		物体材质控制
- v,b		放大/缩小物体
- h		显示/不显示avi播放方格
- *		灯光颜色控制
- 4,6,8,2,7,9	灯光位置控制

# 实现功能
- 视点与墙壁的碰撞检测
- 简单的连续帧绘制（电风扇）
- obj格式文件的导入
- bmp格式纹理的绘制
- 场景漫游及观察
- 鼠标的拖拽控制
- 室内场景的搭建
- 截图功能
- 物体的添加功能
- 物体的平移及放大缩小功能
- 物体材质的修改功能
- 灯光效果
- 灯光位置的控制及颜色的控制
- nurbs曲面的表达
- avi动画的内嵌播放

# 注意事项
该项目仍需要很多优化的地方，运行时文件关系依赖可能出错，且请保证电脑运行状况良好，性能不差，否则容易崩溃
